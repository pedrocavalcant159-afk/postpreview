COLE APENAS ESTAS REGRAS NO FIRESTORE
Firebase Console > Firestore Database > Regras

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /shares/{shareId} {
      // Quem recebe o link só pode ler prévias já finalizadas.
      allow read: if resource.data.public == true;

      // Cada pessoa anônima cria apenas os próprios links.
      allow create: if request.auth != null
                    && request.resource.data.ownerId == request.auth.uid;

      allow update, delete: if request.auth != null
                            && resource.data.ownerId == request.auth.uid;

      // Cada imagem é um documento separado, dentro da prévia.
      match /images/{imageId} {
        allow read: if get(/databases/$(database)/documents/shares/$(shareId)).data.public == true;

        allow create, update, delete: if request.auth != null
          && get(/databases/$(database)/documents/shares/$(shareId)).data.ownerId == request.auth.uid;
      }
    }
  }
}

IMPORTANTE
- Esta versão NÃO usa Firebase Storage.
- Não precisa mexer em regras do Storage.
- Mantenha Authentication > Sign-in method > Anônimo ativado.
