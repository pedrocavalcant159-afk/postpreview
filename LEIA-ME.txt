VERSÃO FIRESTORE GRÁTIS — LINK DE APROVAÇÃO

1. Suba index.html e firebase-config.js para a raiz do GitHub Pages.
2. Em Firebase Console > Firestore Database > Regras, substitua as regras pelo conteúdo de REGRAS-FIREBASE.txt e publique.
3. Mantenha Authentication > Sign-in method > Anônimo ativado.
4. Não é necessário usar Firebase Storage.

Como funciona:
- Ao clicar em “Gerar link”, o site comprime os cards no navegador.
- Cada card é salvo em um documento separado no Firestore.
- O link abre somente a visualização, sem edição.

Limites práticos desta versão:
- Até 10 cards.
- As imagens são convertidas para JPG e comprimidas para manter cada documento abaixo do limite do Firestore.
- Para preservar qualidade, prefira artes de até 1600 px no lado maior.
